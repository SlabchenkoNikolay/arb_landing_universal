# Landing Page — PHP + JS (demo)

Учебный/портфолио-проект: одностраничный лендинг с формой заказа, трекингом и интеграциями. Тематика обезличена; в репозитории только код и нейтральные тексты.

## Стек

- **Backend:** PHP (без фреймворков), конфиг + функции + локализация + обработчик формы
- **Frontend:** HTML5, CSS, JavaScript (jQuery), intl-tel-input для телефона
- **Интеграции:** UTM/subid-трекинг, Facebook Pixel (опционально), передача лидов во внешние системы (postback/API)

## Структура

| Файл / папка | Назначение |
|--------------|------------|
| `config.php` | Конфигурация: гео, продукт (название/цена/валюта), трекинг, безопасность, логи |
| `functions.php` | Вспомогательные функции: IP, геолокация по IP, определение страны |
| `localization.php` | Тексты интерфейса и отзывов по языкам (ru, hi, it, custom) |
| `index.php` | Главная страница: видео-блок, форма заказа, комментарии, подключение скриптов |
| `handler.php` | Обработка POST-формы: валидация, антиспам (honeypot, timestamp), постбек/API, редирект на success |
| `success.php` | Страница после успешной отправки формы |
| `main.css` | Стили лендинга |
| `script.js`, `video-handler.js`, `comments-carousel.js`, `phoneCheck.js`, `time_stat.js` | Клиентская логика |
| `logs/` | Логи (в .gitignore; не коммитить реальные логи) |

## Запуск

1. PHP 7.4+ с расширениями: curl, json, mbstring
2. Положить проект в корень веб-сервера или в подпапку
3. Создать папку `logs/` с правами на запись
4. Видео и картинки (product.png, img/, channel_pics/) в репо не включены — подставь свои или заглушки

## Безопасность и валидация

- Сбор GET-параметров с `htmlspecialchars`
- Honeypot-поле и проверка времени отправки формы
- Ошибки PHP пишутся в лог, не в вывод
- Проверка дубликатов лидов по номеру телефона (файл в `shared/`)

## Лицензия

Код для портфолио/учебных целей. Тексты и бренды обезличены.
